Chrome resize options

This simple extension is designed for very quick flicking between set media queries. Click on a width or fullscreen/maximised to resize the browser.

Changing widths

You can change the widths from the 'Change widths' button and adding the values as a comma seperated list, the default is set to '320,640,960,maximized,fullscreen'

To create the extension without adding any permissions the difference between the content width and the browser actual width is set manually.

The default for this is set to 16px (this figure is from Chrome on Windows 7).

You can test for difference by using the test html page in the github repo and checking the media query values match the widths calculated.
